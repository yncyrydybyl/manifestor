What it does

⚡

One keystroke

Type m. Your latest download lands in the current directory with a clean name.

🧹

Clean names

Screenshot 2026-04-05 at 14.32.07.png becomes screenshot-2026-04-05-at-14.32.07.png

🛡️

Staleness guard

If the newest download is over 8 hours old, asks for confirmation. Skip with mm.

🎨

9 animations

Every copy plays a random manifestation animation. Rainbow beams, portals, forging in flames.

🔒

Safe by default

Same-file guard prevents data loss. Staleness check catches stale grabs. Force mode when you mean it.

🍺

Brew ready

brew install, completions auto-installed, mm symlink set up. Zero config.

Install

Homebrew

brew install yncyrydybyl/tap/manifestor

From source

git clone https://github.com/yncyrydybyl/manifestor.git
cd manifestor
make install

Pick an animation

# see all animations
m --list-anims

# use a specific one
m --anim fire-forge

# skip animation
m --no-anim

Binary download

# Check releases for your platform:
# github.com/yncyrydybyl/manifestor/releases
curl -sL [release-url] -o m
chmod +x m
sudo mv m /usr/local/bin/
